# Pay Offer 1.1.0

Возможность покупать платные размещения предложений для Project, неограниченое количество тарифов, скидки для PRO, количество бессплатных предложений в день, возможность в администрировании управлять количеством доступных пользователю предложений.

### Установка:

1. Загрузить и распаковать файли в папку plugins.
2. Установить через панель: (Управленние сайтом / Расширения / Купить "Предложения").
3. Настроить конфигурации и по необходимости добавить ссылку на страницу покупки в header.tpl {HEADER_USER_PAYOFFER}
	Форму добавления предложения ( ADDOFFERFORM ) можно заключить в условие  <!-- IF {OFFER_FORM_PO_COUNT_LEFT} > 0 --> в ином случаи если не доступно опубликовать - будет ошибка.
	Для информирования пользователя можно добавить такой блок кода в project.offers.tpl 
	<!-- IF {OFFER_FORM_PO_INFO} -->
		<div class="alert">{OFFER_FORM_PO_INFO}. {OFFER_FORM_PO_BUY_URL}</div>
	<!-- ENDIF -->
